The B.A.L.L.E.R.

Built by team members and I, the Balancing Automated Leveraging and Levelling Electronic Robot (B.A.L.L.E.R.), as is its name suggests, balances a 1 inch steel bearing ball on its surface through any disturbance. This project was was helped with open source files provided by Aaed Musa. About 40 hours were needed to complete this project using an Arduino with C++ code.

Self Web Hosting

Display Board

For a competitive biology project in grade 12, I made a cross section of the human body board and put programmable LEDs at the back to illustrate the flow of nutrients through the digestive system. A button was put on the box to activate the animation and a liquid-crystal display to show text for each section of the system. I custom built the 3D printed box using Fusion360 to fit electonic components such as the Arduino. This project won first place in the competition.
